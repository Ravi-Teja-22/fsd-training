 2) Key Concept: Feature-Based Architecture (Most Important)

Instead of grouping by file type, companies group by **features/modules**.

Example:


features/
  auth/
  users/
  products/
  orders/


Each feature contains:

* UI (components, pages)
* State (Redux slice)
* API calls
* Business logic
