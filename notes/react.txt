traditional react commands
npm- node package module

1. npx create-react app app-name = creates a react applications with that app name.
2. npm start = runs the application.

disadvantage with traditional method:
- loads and runs very slow, time taking process.

vite method commands
1.npm create vite@latest . = creates a react app with vite method.
2. npm i = installs all the libraries required.
3. npm run dev = runs the application.

react hooks - hooks are special js functional components to use features like state life cycle methods, and more features that were earlier available only in class components.
